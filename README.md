# 🎮 Projet NSI : Jeu de Plateforme

Bienvenue dans mon projet de fin d'année de **NSI (Numérique et Sciences Informatiques)**. Il s'agit d'un jeu de plateforme développé entièrement en Python.

## 🚀 Comment jouer ?

### Option 1 : Version déjà compilée (Recommandé)
Si vous voulez jouer sans installer Python :
1. Allez dans l'onglet **[Releases](Lien_Vers_Tes_Releases_Ici)** sur GitHub.
2. Téléchargez le fichier `Mario_Maker.zip`.
3. Faites un clic droit sur le fichier et choisissez **Extraire tout**.
4. Lancez l'application nommée `programme_principal.exe`.

### Option 2 : Lancer à partir du code source
Si vous avez Python installé sur votre machine :
1. Clonez ce dépôt ou téléchargez les fichiers.
2. Installez la bibliothèque nécessaire :
   ```powershell
   pip install pygame
